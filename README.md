# Proyecto Validando hipotesis de negocio con pruebas estadisticas
## 🎯 Objetivo del Proyecto

El objetivo principal del proyecto es evaluar y validar estadísticamente un experimento A/B realizado en la página de inicio (landing page) de un e-commerce para determinar qué versión (A o B) optimiza la tasa de conversión y el valor económico por usuario, permitiendo tomar una decisión de negocio basada en datos.

## 📊 Conjuntos de datos utilizados

1. landing_experiment.csv: Información de los usuarios expuestos a las versiones A y B del experimento (incluye ID de usuario, fecha, versión de la página, región, dispositivo, fuente de tráfico, tipo de usuario, si convirtió y el monto gastado).

## 🛠️ Etapas del Análisis

1. Cargar y validar los datos
2. Comparar el gasto promedio por usuario (página A vs. B)
3. Comparar la tasa de conversión entre la página A y B
4. Analizar la relación entre la fuente de tráfico y la conversión
5. Analizar la relación entre el tipo de usuario y la conversión
6. Visualización de resultados y conclusiones finales

## 🚀 Cómo ejecutar el cuaderno
